Sobre o projeto

Angular University

Catálogo mundial de universidades, construído com Angular para consumir uma API REST pública e exibir dados reais de instituições de ensino superior de qualquer país do mundo.

flag Objetivo

Este projeto foi desenvolvido como atividade acadêmica da disciplina de Desenvolvimento Web com Angular, aplicando na prática conceitos de componentes, serviços, interfaces, rotas, formulários, data binding e consumo de APIs REST.

public Fonte dos dados

Os dados são fornecidos pela Universities API (Hipolabs), uma API pública, gratuita e sem necessidade de autenticação.
